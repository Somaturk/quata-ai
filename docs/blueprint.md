# **App Name**: ProVision

## Core Features:

- Navigation Bar: Implement the top navigation bar with the following items: Teklif (Offer), Ürün Yükle (Upload Product), Ürünler (Products), Firmalar (Companies), Eski Teklifler (Old Offers), Siparişler (Orders), Görüşmeler (Meetings)
- Offer Section: Design the Offer section (Teklif) with fields such as Teklif No (Offer Number), Tarih (Date), Geçerlilik Tarihi (Expiration Date).
- Product Upload: Implement product upload functionality where users can add product details such as name, description, and pricing.
- Product Management: Enable users to manage a list of products with details, including editing and deletion options.
- Company Management: Allow users to manage company information including names, addresses, and contact details.

## Style Guidelines:

- Primary color: Dark teal (#008080), taken from the headings and used to provide a professional and modern feel.
- Background color: Light grayish-blue (#F0F8FF) to ensure readability and a clean interface.
- Accent color: Light blue (#ADD8E6), slightly lighter shade of blue to highlight interactive elements.
- Body and headline font: 'Inter', sans-serif font, to maintain the text's legibility.
- Note: currently only Google Fonts are supported.
- Ensure a clean, intuitive layout inspired by the image, with clear sections for data input and display.
